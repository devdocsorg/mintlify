# Creating a review team in Public Groups

## Prerequisites

- The Group members must be already configured

## Steps

### 1) Navigate to Setup

- Log in to Salesforce.
- Click on the **Setup** icon (gear icon) in the top right corner.

![Click on Gear Icon](/snippets/smarteeva-screenshots/use-setup-quickfind-01-click-on-gear-icon.jpg)

### 2) Access Public Groups

- Access the **Quick Find** box

![Access Quick Find](/snippets/smarteeva-screenshots/use-setup-quickfind-03-click-on-setup-quick-find.jpg)

- Type **Public Groups**.

![Enter Public Groups](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-01-enter-public-groups.jpg)

- Select **Public Groups** under the **Users** section.

![Click on Public Groups](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-02-click-on-public-groups.jpg)

### 3) Create a New Public Group

- Click the **New** button.

![Click New](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-03-click-new.jpg)

### 4) Define Group Details

- **Label**: Enter a name for the group (e.g., `Complaint Review Team`).

![Set Label](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-05-set-label-complaint-review-team.jpg)

- **Group Name**: This will auto-populate based on the label.

![Autofilled API Name](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-06-autofilled-api-name.jpg)

### 5) Set Group Membership

- In the **Search** section, select the type of members (e.g., Users, Roles, Public Groups).

![Set Users](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-07-search-set-users.jpg)

- Use the search box to find and select members.

![Search Users for Test User](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-08-search-users-for-test-user-click-find.jpg)

- Move selected members to the **Selected Members** box.

![Add Test User](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-09-available-test-user-click-add.jpg)

### 6) Save the Group

- Click **Save** to create the group.

![Click Save](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-10-test-user-selected-click-save.jpg)

### 7) Verify Group Creation

- Ensure the group appears in the **Public Groups** list.

![Verify Group Listed](/snippets/smarteeva-screenshots/creating-smarteeva-review-team-public-group-11-listed.jpg)
