# Using Template Tags

## Introduction



Smart Record Templates help streamline data entry in Smarteeva's Compliance Toolkit. Template Tags can further improve this process by categorizing templates and making them easier to find and apply. This guide explains how to use Template Tags effectively.

## Understanding Template Tags



Template Tags are keywords or phrases that categorize Smart Record Templates. They help manage and filter templates, especially when dealing with a large number of them.

### Benefits of Using Template Tags

- Easier template filtering
- Quicker template selection
- Better template organization

### Understanding Tag Matching Behavior

You can configure how templates are filtered based on tags:

- Templates with at least one matching tag (OR logic)
- Templates with all matching tags (AND logic)

## Best Practices for Using Template Tags



#### Consistent Tagging
Establish a standard for tags across templates.

#### Use Relevant Tags
Assign meaningful tags that help in effective filtering.

#### Leverage Automatic Tagging
Use merge fields to reduce manual filtering.

#### Avoid Over-Tagging
Keep tags concise to maintain effective filtering.

#### Test Configurations
Ensure templates are filtered as expected with various records.

## Conclusion



Template Tags can significantly improve the efficiency of template management and application in Smarteeva. By implementing the practices outlined in this guide, you can streamline your workflow and improve data entry processes.
