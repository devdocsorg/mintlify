# Regulatory Resources

## FDA Resources

| Resource Name                                                                                                                                                                                                   | Description                                                                                                                                                             |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [MedWatch Forms](https://www.fda.gov/safety/medical-product-safety-information/medwatch-forms-fda-safety-reporting)                                                                                             | Provides various MedWatch forms for reporting medical device safety issues to the FDA, including Form 3500A for voluntary reporting by healthcare professionals         |
| [MDR Data Files](https://www.fda.gov/medical-devices/mandatory-reporting-requirements-manufacturers-importers-and-device-user-facilities/about-manufacturer-and-user-facility-device-experience-maude-database) | Offers downloadable data files containing historical MDR (Medical Device Report) data from 1996 onwards for analysis and research purposes                              |
| [MDR Adverse Event Codes](https://www.fda.gov/medical-devices/mandatory-reporting-requirements-manufacturers-importers-and-device-user-facilities/mdr-adverse-event-codes)                                      | Explains the structure and definitions of the FDA's Medical Device Report (MDR) adverse event codes for classifying medical device problems, causes, and health effects |
| [MAUDE Database Search](https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfmaude/search.cfm)                                                                                                                  | Allows users to search the Manufacturer and User Facility Device Experience (MAUDE) database for specific medical device reports submitted to the FDA                   |

## NCI Thesaurus Resources

| Resource Name                                                                             | Description                                                                                                                                    |
| ----------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| [NCI Thesaurus (NCIt) Browser](https://ncithesaurus.nci.nih.gov/ncitbrowser/)             | Official browser for exploring and searching the NCIT terminology, including terms, hierarchical relationships, definitions, and other details |
| [NCI Thesaurus (NCIt) FTP Site](https://evs.nci.nih.gov/ftp1/NCI_Thesaurus/)              | Provides downloadable files of the NCIT terminology in various formats for integrating NCIT into local systems or databases                    |
| [NCI Thesaurus GitHub Repository](https://github.com/NCI-Thesaurus/thesaurus-obo-edition) | NCI Thesaurus available on GitHub in the Open Biomedical Ontologies (OBO) format, providing version-controlled files and issue tracking        |
