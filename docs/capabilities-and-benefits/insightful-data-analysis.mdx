# Insightful Data Analysis

Smarteeva transforms raw data into actionable insights through its Meaningful Data Change (MDC) functionality. By
highlighting significant changes, facilitating collaborative reviews, and integrating with Salesforce's reporting
capabilities, users can efficiently monitor critical information and identify trends. From inline record views to custom
dashboards, Smarteeva offers flexible tools to analyze data in ways that drive informed decision-making.

<div className="docs-icon-heading">
  <Key className="docs-icon" />
  ## Key Features
</div>

- **[Meaningful Data Change (MDC) Reports](/docs/packages/cap/reference/meaningful-data-change-reports):** Generate reports that highlight significant changes to records based on
  pre-defined criteria for specific user groups.
- **MDC Review Functionality:** Allow designated users to review and mark MDC report entries as reviewed, streamlining
  collaboration and ensuring accountability.
- **MDC on Record (Inline View):** Display summarized MDC reports directly on record pages, providing a quick overview
  of meaningful changes.
- **Customizable Dashboards and Reports:** Leverage Salesforce's reporting and dashboard capabilities to create custom
  visualizations and analyze data trends.
- **Integration with External Systems (Via Salesforce APIs):** Connect Smarteeva data with other business intelligence
  tools for advanced analytics and reporting.
